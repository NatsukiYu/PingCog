== 詳細な設定

=== guild_ids
Slash CommandsはデフォルトでGlobal Commandsとして扱われます．
Guild Commandsとしたいときは，環境変数 `guild_ids` を設定してください．
例． `guild_ids=11111111,22222222`

[IMPORTANT]
====
`import ping_cog` の時点で環境変数が参照されます．
`dotenv` などはインポートの前に行ってください．
====
